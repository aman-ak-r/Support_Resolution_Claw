# Support Resolution Claw 🦅

An autonomous AI agent pipeline built with **LangGraph**, **FAISS**, and **Sentence-Transformers** to assist micro-entrepreneurs and merchant partners. The system semantically searches a local FAQ/SOP knowledge base and either resolves the query instantly or escalates it into a structured ticket log using custom scoring algorithms.

---

## Key Features
- **LangGraph Routing Workflow**: Uses a stateful graph to guide queries through retrieval, evaluation, and resolution or escalation.
- **Hybrid Confidence Score**: Blends FAISS vector distance and LLM semantic completeness rating to prevent incorrect answers.
- **Automatic Multi-Provider Support**: Swappable integration for OpenAI, Gemini, and Anthropic. Includes a **Mock LLM mode** out of the box requiring zero API keys.
- **Relational Logging**: Every interaction is logged to a local SQLite database file.
- **Interactive UI**: Streamlit dashboard with live telemetry, operational statistics, ticket viewers, and database exploration.

## Quick Setup

### 1. Prerequisites
Ensure you have **Python 3.9 to 3.11** installed.

### 2. Install Dependencies
Run the following command to install the required libraries:
```bash
pip install -r requirements.txt
```

### 3. Environment Configuration
By default, the project runs in `Mock` mode which requires no API key. To connect to active cloud model providers, rename `.env.example` to `.env` and fill in the values:
```bash
# To run with OpenAI:
LLM_PROVIDER=openai
OPENAI_API_KEY=your-api-key

# To run with Gemini:
LLM_PROVIDER=gemini
GEMINI_API_KEY=your-api-key
```

## Running the Application

### 1. Build the FAISS Vector Index
Before the app can perform retrieval, it must index the markdown knowledge base. You can build it by running:
```bash
python -c "from agent.retrieval import KnowledgeBaseRetriever; r=KnowledgeBaseRetriever(); r.build_and_save_index()"
```
*(Alternatively, you can click the **⚙️ Rebuild Vector Index** button inside the Streamlit sidebar at any time to compile/refresh it).*

### 2. Run the Streamlit Dashboard
Launch the web interface:
```bash
streamlit run app.py
```
Open the browser at `http://localhost:8501` to use the application.
