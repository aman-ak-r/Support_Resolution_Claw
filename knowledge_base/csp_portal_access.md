# SOP: CSP Portal Access and Login Issues

Troubleshooting access errors, IP locks, and browser compatibility for Eko Customer Service Points.

## Resolutions
1. **IP Lock Error**: CSP access is geo-restricted. Ensure GPS is enabled on mobile or use verified static broadbands.
2. **Credential Locked**: 3 incorrect login attempts block access for 2 hours. Reset using 'Forgot Password'.
