# FAQ: Handling Network and Server Connection Errors
    
Q: The Eko app displays "Network Error" or "Server Unavailable". How to fix?
A: Check the following steps:
1. Ensure your internet connection is stable. Switch between mobile data and Wi-Fi to verify.
2. Clear the cache of the Eko App in your Android Settings -> Apps -> Eko -> Storage -> Clear Cache.
3. Check if there is an active VPN running on your device; VPNs block Eko's API security gateways.
4. If Eko servers are undergoing maintenance, a banner will be displayed. Wait for maintenance to finish.
