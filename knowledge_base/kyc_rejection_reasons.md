# FAQ: Reasons for KYC Rejections
    
KYC verification might be rejected for several key reasons:
1. Document text is blurry or unreadable.
2. Uploading photocopies or scanned prints instead of original documents.
3. The address on Aadhaar does not match the shop address (for out-of-state registrations, additional rent agreements or utility bills are required).
4. The uploaded selfie does not match the photo on Aadhaar or PAN.

If rejected, you will receive a notification explaining the exact issue. Correct the specified documents and upload them again.
