# SOP: Handling Delayed or Missing Commissions
    
If your real-time commission is not credited:
1. Wait 30 minutes, as system sync delays can sometimes occur during high-traffic windows.
2. Check the transaction log status. If the parent transaction shows 'Pending', commission will only be credited when status turns 'Success'.
3. If the transaction was 'Success' but commission is missing, check if the payout was adjusted against active TDS deductions or other recovery balances.
4. Raise a support ticket using the category 'Commission -> Delayed Crediting' with the Transaction ID (TXNID).
